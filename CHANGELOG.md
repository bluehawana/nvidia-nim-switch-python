# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-02

### Added
- Initial release of cc-nvd-python
- Dynamic model switching between NVIDIA NIM models
- Web interface for browsing and selecting models
- RESTful API for programmatic model management
- Persistent model selection across restarts
- One-line installation script
- Homebrew formula support
- PyPI package distribution support
- Comprehensive documentation
